# react-native-uniffi-russh

Uniffi bindings for russh

## Installation

```sh
npm install react-native-uniffi-russh
```

## Usage

```js
import { multiply } from 'react-native-uniffi-russh';

// ...

const result = multiply(3, 7);
```
