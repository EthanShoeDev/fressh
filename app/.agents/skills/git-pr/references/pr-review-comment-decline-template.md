# PR Review Comment Decline Template

Use this when choosing not to implement a PR review suggestion and resolving the thread.

```md
Not implementing this in this PR; resolving this thread.

Reason:
- <why this is not needed/appropriate now>

Context:
- <tradeoff, constraint, or existing behavior that covers it>

Follow-up:
- <issue link for later work, or "none">
```

Rules:

- Always explain why before resolving the thread.
- Resolve only after the rationale comment is posted.
- If the suggestion is valid but out of scope, open/link a follow-up issue.
